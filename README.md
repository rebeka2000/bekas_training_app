# Bekas Training App

A web prototype training planner for running, yoga, Pilates, walking, incline walking, stair stepper, strength, mobility, recovery, and lifestyle habits.

The app is designed around Bekas Training App: a soft pastel coaching experience inspired by structured running plans and readiness/lifestyle coaching, with flexible 1, 2, or 3 training sessions per day.

## Features

- Weekly planner with per-day session count
- Morning, daytime, and evening training windows
- Custom duration per session
- Multiple training types: run, yoga, Pilates, walking, incline walking, stair stepper, strength, mobility, and recovery
- Daily sleep, stress, and energy check-in
- Readiness, recovery, training density, streak, and consistency views
- Simple AI-coach style adjustment chat

## Run locally

Open `index.html` in a browser, or serve the folder locally:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173`.
